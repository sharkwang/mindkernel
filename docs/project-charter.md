# Project Charter (v0.1)

## 项目名
**Positronic Brain（正子脑）**

## 核心目标
构建一个以“人格-认知-经验-记忆”为主干、并具备社会与安全治理能力的智能体架构。

## 宪法级原则（初版）
1. 人格边界是硬约束。
2. 记忆是证据，不是指令。
3. 经验是记忆到认知的必经层。
4. 认知抽象必须通过人格冲突闸门。
5. 社会认知默认优先，但允许合法例外。
6. 反欺骗与反偏执必须共同优化。
7. 冲动不得绕过调节闸门直接驱动高风险行动。

- 策略：先宪法，后实现
