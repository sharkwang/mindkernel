# positronic-brain

> 项目名称来源：Isaac Asimov 机器人系列中的 **Positronic Brain（正子脑）**。

## 为什么改名
我们当前讨论已经超越“记忆系统优化”，进入更完整的智能体心智架构：

- 人格（Persona / Soul）
- 认知（Cognition）
- 经验（Experience）
- 记忆（Memory）
- 情绪-冲动-行动（Affect-Impulse-Action）
- 社会认知与治理机制（Social/Governance）

因此采用更上位、更中性且有历史语义的名称：**positronic-brain**。

## 项目定位
本项目用于定义并实现一个“可演化、可审计、可约束”的智能体心智系统。

## 目录
- `docs/`：新版本设计文档（清晰、分层、可执行）
- `archive/`：从 `memory-v2` 迁移来的历史草案（保留上下文）
- `schemas/`：当前生效的数据契约
- `data/`：样例/评估数据与实验产物

## 迁移说明
已将 `memory-v2` 的核心文件迁移到 `archive/`，作为历史输入，不直接作为新规范。
