# memory-v2

记忆系统改进项目（Memory v2）。

## 目标
- 让“聊过的事”稳定落盘
- 让长期记忆可追溯（有来源）
- 让记忆维护可周期化执行

## 目录结构
- `docs/`：设计文档与决策记录
- `schema/`：记忆条目结构定义
- `scripts/`：采集、提炼、校验脚本
- `data/raw/`：原始候选记忆（自动采集）
- `data/curated/`：人工确认后的记忆
- `data/index/`：检索索引或中间产物
- `tests/`：召回/准确率回归测试

## 初始里程碑
1. 定义 Memory Entry Schema
2. 定义 capture -> curate -> publish 工作流
3. 做一轮历史记忆迁移样例
4. 建立最小回归测试集
